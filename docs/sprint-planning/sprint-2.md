## Product Backlog Seleccionado

### Historias de Usuario Comprometidas

#### Historia #2: Login de iniciar sesión
- **Como:** cliente nuevo o recurrente
- **Quiero:** registrarme e iniciar sesión con mi cuenta (email y contraseña)
- **Para:** acceder a mi perfil y realizar compras de manera personalizada.
- **Story Points:** [4]
- **Criterios de Aceptación:**
  - [ ] Registrarse ingresando email y contraseña.
  - [ ] El sistema valida que el correo esté registrado.
  - [ ] El sistema valida que los campos no estén vacíos.
  - [ ] El sistema muestra error si los datos son incorrectos.
  - [ ] Debe existir opción de "Olvidé mi contraseña".
